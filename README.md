# Jungyeon Workspace

> 서울대학교 AI 공학연구원 로봇 기술혁신센터
> 서정연 연구원 개인 작업공간

## 개요
산업통상자원부 국가과제(2025~2029) 로봇 기술혁신센터 업무 중,
휴머노이드 로봇 및 로봇 핸드 도입 검토를 위한 개인 작업 저장소.

## 폴더 구조
- `휴머노이드 로봇 및 로봇 핸드 관련/` — 견적서·비교표·제품 매뉴얼
- `간판 관련/` — 해동첨단공학관 간판 설치 관련 문서
- `CLAUDE.md` — AI 에이전트용 프로젝트 컨텍스트

## 진행 현황 (2026-04-15 기준)

### 견적서 수령
- 원익로보틱스 Allegro Hand V5 (4F)
- 테솔로 DG-5F-M / DG-5F-S
- 유니트리 G1 Edu Ultimate(D)-U6 (위고로보틱스)

### 견적서 대기
- Rainbow Robotics, ROBOTIS, 로브로스, 홀리데이 로보틱스
