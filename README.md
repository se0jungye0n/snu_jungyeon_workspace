# Jungyeon Workspace

> 서울대학교 AI 공학연구원 로봇 기술혁신센터
> 서정연 연구원 개인 작업공간

## 개요
산업통상자원부 국가과제(2025~2029) 로봇 기술혁신센터 업무 중,
휴머노이드 로봇 및 로봇 핸드 도입 검토를 위한 개인 작업 저장소.

## 폴더 구조
- `휴머노이드 로봇 및 로봇 핸드 관련/` — 견적서·비교표·제품 매뉴얼·견적요청서
- `센터 공사/` — 간판·디스플레이·인테리어 관련 문서
- `규정, 서식, 참고자료/` — 과제 관련 규정·서식·절차 문서
- `CLAUDE.md` — AI 에이전트용 프로젝트 컨텍스트
